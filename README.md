## Plugins

### Vanguard

Контекстная панель модерации для тегов, заметок и быстрых действий администратора в **exteraGram**.

### Docky
A dock for favorite chats inside conversations, allowing quick access and preview without leaving the current chat.

**RU:**  
Док для избранных чатов внутри диалогов, позволяющий быстро открывать и просматривать чаты, не покидая текущий.

[Download plugin](#)

---

### Nowfy
Displays and controls the current playing track with a styled preview inside **exteraGram**.

Supported services:
- Spotify (API)
- Last.fm (scrobbling)
- stats.fm (Spotify and Apple Music tracking)

**RU:**  
Отображает и управляет текущим воспроизводимым треком с оформленным предпросмотром внутри **exteraGram**.

Поддерживаемые сервисы:
- Spotify (API)
- Last.fm (скробблинг)
- stats.fm (отслеживание Spotify и Apple Music)

[Download plugin](#)

### ChatList Customizer
Enhances the chat list with dynamic time-based greetings, customizable titles, Now Playing integration and real-time weather updates.

**RU:**  
Расширяет список чатов динамическими приветствиями по времени суток, настраиваемыми заголовками, интеграцией Now Playing и отображением погоды в реальном времени.  
Поддерживает **stats.fm** и **Last.fm**.
